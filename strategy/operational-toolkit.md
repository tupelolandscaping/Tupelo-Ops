# 3. Quoting Formula (+ Homeworks Rate Matrix)

**Formula:**** **Quote = estimated person-hours × 1.30 buffer × rate

- **Rates:**** **$40/hr hand-tool basis; $50/hr existing power/mowing; $60/hr new mowing clients (2027).

- **The 1.30 buffer**** **corrects the observed ~30% time underestimate on project work and protects margin. Apply it to project quotes now.

**Worked example:**** **A project estimated at 12 person-hours × 1.30 × $50 = $780 (vs. the $600 an unbuffered quote would have produced — the difference is recovered margin, not padding).

## Encode it in Homeworks

- Build the rule in **Rate Matrix Pricing** so the lead or Anais can quote without your judgment — the goal is “anyone can be an estimator.”

- As **Job Costing** data accumulates, tune the 1.30 buffer downward toward your true estimate accuracy. The buffer is a temporary correction, not a permanent tax.

# 5. Lead-Capture Intake (Homeworks Setup)

Capture every contact so no spring prospect evaporates. A 20-second entry per door is the whole mechanism; the discipline of recording is the hard part, not the tooling.

## Fields to record

- Date | Address | Name (if given) | Phone/email (if given) | Source (door / sign / referral / online) | Outcome tag | Follow-up date | Notes

## Outcome tags

| **Tag** | **Use / follow-up** |
| --- | --- |
| Interested – spring | Wants service next season → follow-up Feb 2027 |
| Has-service – declined | Already has a provider → follow-up Feb 2027 (often the best spring conversion pool) |
| One-off project | Project lead → follow-up by project type |
| Booked | Converted — move to client record |
| Not interested | Suppress |

## Where in Homeworks

- Use the **Waitlist** and embeddable **Work-request form** for inbound; log canvassing outcomes as lead records.

- In Feb 2027, filter **Interested-spring + Has-service-declined** and fire the automated SMS/email campaign — executable remotely.

# 8. Labor Compliance Checklist (VA / Federal)

# 9. Referral Program Spec

Your best owner-independent channel: referred clients convert best, cost almost nothing, and the system runs without you. With ~$890 lifetime profit per recurring client, a reward of $25–50 per side is trivially profitable.

## Structure

- **Give-get:**** **an existing client refers a neighbor; both receive a [$25–50] credit (or one free mow) once the new client completes their first visit.

## Mechanics (Homeworks)

- Issue and track rewards via **Customer Credits**.

- Use **Automations** to request a referral right after a high rating or 5-star review — when goodwill peaks.

- Put a referral QR code on yard signs, invoices, and the truck.

## Sample client-facing copy

*“**Love your lawn? Refer a neighbor — you both get $[__] off when they sign up. Just share this link/code.**”*
